## Visual WinUI3
A tool to create a WinUI3 Visually. Supports export to XAML and real-time preview of the UI.

## Supported elements so far
* UIElement
  * FrameworkElement
    * TextBlock
    * Control
      * ButtonBase
        * Button
        * HyperlinkButton
        * ToggleButton
        * CheckBox
      * TextBox
      * RatingControl
    * Panel
      * StackPanel
