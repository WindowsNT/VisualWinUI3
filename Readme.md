## Visual WinUI3
A tool to create a WinUI3 Visually. Supports export to XAML and real-time preview of the UI.
Supported elements don't include all their attributes - work is in progress.

## Supported elements so far
* UIElement
  * FrameworkElement
    * Control
      * ButtonBase
        * Button
        * CalendarDatePicker
        * CalendarView
        * CheckBox
        * DatePicker
        * HyperlinkButton
        * TimePicker
        * ToggleButton
      * Image
      * InfoBar
      * ItemsControl
        * Selector
          * FlipView
          * GridView
      * MediaPlayerElement
      * ProgressBar
      * ProgressRing
      * RatingControl
      * TextBox
    * Panel
      * StackPanel
      * Grid
    * TextBlock
    * WebView2
